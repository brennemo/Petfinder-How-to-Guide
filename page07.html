Adding a Button and Event Listener

Last, but not least, we'll add a button, so that the user can get a new random cat without having to refresh the whole page from the browser. In our HTML file, below the table, we've added a button labeled "See more!" that has an ID of randomcat.

In the function buttonClick, first, any remaining images are removed, since our original function appends them to the table cell. Next, the same requestJSONP function is called, to fill the table with the information of another random cat. Finally, we add an Event Listener to the randomcat button, that takes the "click" event and the buttonClick function as parameters. Now, upon clicking the button, the photo and basic profile of a new cat is loaded!
